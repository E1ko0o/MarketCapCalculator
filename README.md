# MarketCapCalculator
Калькулятор для подсчета акций по таким критериям, как:
* P/E
* P/S
* P/B
* EPS
* EBITDA
* Total debt
* Net income
* ROE
* ROA
* Market cap

Для использования нужен интернет, так как получение актуальных данных происходит с yahooFinance.

Каждая компания обрабатывается примерно 12 секунд (12000 миллисекунд) - количество анализируемых компаний изменяется, вы можете увидеть и изменить список компаний в файле tikers.csv.

Причина отказа от распределения по капитализации: при средней цене акции и малой капитализации невозможно подобрать оптимальное количество акций - из-за этого остается много невостребованных денег.
